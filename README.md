# คำนวณค่า AVE และตรวจสอบค่าสัมประสิทธิ์สหสัมพันธ์

โปรเจกต์นี้ใช้สำหรับคำนวณค่า AVE ของตัวแปรต่างๆ และตรวจสอบว่าค่า AVE นั้นมากกว่าค่าสัมประสิทธิ์สหสัมพันธ์ยกกำลังสองระหว่างตัวแปรนั้นกับตัวแปรอื่นๆ หรือไม่

## โครงสร้างไฟล์

- `index.html` - ไฟล์ HTML หลักที่ประกอบด้วยฟอร์มสำหรับใส่จำนวนตัวแปร, ค่า AVE และค่าสัมประสิทธิ์สหสัมพันธ์

## การใช้งาน

1. เปิดไฟล์ `index.html` ในเว็บเบราว์เซอร์
2. ระบุจำนวนตัวแปรในฟอร์มและกดปุ่ม "สร้างตาราง"
3. ใส่ค่า AVE และค่าสัมประสิทธิ์สหสัมพันธ์ระหว่างตัวแปรต่างๆ ในตารางที่ปรากฏ
4. กดปุ่ม "คำนวณ" เพื่อดูผลการคำนวณและตรวจสอบค่า AVE
